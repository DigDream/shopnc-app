###命名规则

String字符串格式的放到String.xml

Color颜色的放到colors.xml

Style风格的放到style.xml

###Activity或Fragment命名规则

整体为xxxActivity，继承BaseActivity

整体为xxxFragment

xxx采取驼峰命名规则，见名知义

对应的Layout命名规则为activity_xxx_xxx，对应驼峰的命名。。

###View组件id的命名
TextView tv_xxx_xxx

ImageView iv_xxx_xxx

Button btn_xxx_xxx

其他的自行想象。。
